# webgl_fundamentals_m3

Various 2d math functions used in the WebGL Fundamentals tutorials.

## Usage

A simple usage example:

```dart
import 'package:webgl_fundamentals_m3/webgl_fundamentals_m3.dart';

void main() {
  translate([
    0, .5, 1,
    1, 0, .5,
    .5, 0, .5
  ], 2, 2);
}
```
